Instalación del proyecto
========================

Requisitos Previos:
- Tener instalado Java Development Kit versión 17 o superior.

- (Opcional) Tener un IDE para Java para facilitar la compilación y ejecución.

Pasos para instalar y ejecutar el proyecto:
1. Descargar o clonar el repositorio de Git
   - Si prefiere el repositorio Git, clónelo con:
     git clone https://github.com/Alan-Vazquez/URL-shortener.git
   - Si prefiere el archivo ZIP, descomprímalo en una carpeta local.

2. Abrir la carpeta del proyecto en tu IDE o en consola.

3. Compilar el proyecto
   Con IDE:
   - Importe el proyecto como proyecto Java estándar.
   - Asegúrese de que el Java Development Kit configurado sea 17 o superior.
   - Compile el proyecto.

   Con consola:
   - Abra la consola.
   - Navegue a la carpeta donde están los archivos .java.
   - Ejecute el siguiente comando para compilar todos los archivos:
     javac *.java
     
4. Ejecutar el proyecto
   Con IDE:
   - Ejecute la clase Main.

   Con consola:
   - Ejecute el siguiente comando:
     java Main

5. Uso del proyecto
   - En la ventana que aparece, puede cargar un archivo con URLs usando el botón “Cargar archivo”.
   - Puede agregar URLs manualmente.
   - Puede visualizar el histograma para entender la distribución de URLs en la tabla hash.
   - Al cerrar el programa, los datos se guardan automáticamente.

Archivos de datos:
- El proyecto puede usar un archivo urls.txt o urls.csv para cargar datos.
- Si no tiene ninguno, puedes crear un archivo de texto con URLs.
- Si desea puede usar el archivo HashMap.ser, que se encuentra dentro de la carpeta "Codigo", que ya contiene 170k URLs aproximadamente.
